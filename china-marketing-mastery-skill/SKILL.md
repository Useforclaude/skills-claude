### 3.4.3 ROI Calculation Framework

**Key Metrics:**

**1. GMV (Gross Merchandise Value)**
```

**2. Net Revenue**
```

**3. Total Cost**
```
Total Cost = Pit Fee + Commission + COGS + Shipping + Platform Fee
```

**4. ROI**
```
ROI = (Net Revenue - Total Cost) / Total Cost √ó 100%
```
