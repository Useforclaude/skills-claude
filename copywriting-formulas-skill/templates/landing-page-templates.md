## Landing Page Copy Checklist

### Above the Fold Must-Haves:
```
[ ] Clear headline (benefit-driven)
[ ] Supporting subheadline
[ ] Hero image/video
[ ] Primary CTA button
[ ] Trust elements (logos, stats, badges)
```

### Throughout Page:
```
[ ] Multiple CTAs (every 1-2 scrolls)
[ ] Social proof (testimonials, stats, logos)
[ ] Benefit-focused (not feature-focused)
[ ] Objection handling (FAQ, guarantee)
[ ] Urgency/Scarcity (if authentic)
[ ] Mobile-responsive
[ ] Fast loading (< 3 seconds)
```

### Before Launch:
```
[ ] Test all forms and buttons
[ ] Check mobile layout
[ ] Run through conversion flow
[ ] A/B test headline variations
[ ] Check analytics tracking
[ ] Test on different browsers
```
